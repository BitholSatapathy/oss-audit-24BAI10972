## Project Description

This project is an audit of Python as open-source software. It includes analysis of its origin, license, ethical aspects, Linux footprint, ecosystem, and a comparison with proprietary software.

## Scripts Included

### 1. System Identity Report

Displays system information like kernel version, user, uptime, and OS details.

### 2. Package Inspector

Checks whether Python is installed and displays version and details.

### 3. Disk and Permission Auditor

Analyzes system directories for size and permissions.

### 4. Log File Analyzer

Counts occurrences of keywords (like errors) in log files.

### 5. Manifesto Generator

Generates a personalized open-source philosophy statement.

## How to Run

```bash
chmod +x script1_system_identity.sh
./script1_system_identity.sh
```

Repeat similarly for the other scripts.
